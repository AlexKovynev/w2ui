Locks a single page of the form.
lockPage(page, message)
page integer, page number
message string, message
Returns boolean.

Description

This method will lock specified page of the form by adding a semi-transparent div on top of its content. The user will not be able to interact with the page or its content until the form is unlocked with .unlockPage(). This method will call w2utils.unlock() to perform the locking.

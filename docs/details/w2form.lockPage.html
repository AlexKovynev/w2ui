Locks an individual page in the form and displays a status message.
lockPage(page, [message])
page number, zero indexed page number
message string, optional, message to display
Returns boolean; true if the page to lock could be found, otherwise false.

Description

This method will lock a page in the form by adding a semi-transparent div on top of its content. The user will not be able to interact with the page's content until the page is unlocked with .unlockPage() or .unlock(). This method will call w2utils.lock() to perform the locking.

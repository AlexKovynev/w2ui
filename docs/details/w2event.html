A class that all w2ui widgets are extended from.
Object
This class provides basic event functionality and only has one property and three methods:
  • handlers, - an array of all event handlers defined for the class
  • on(event, handler) - method to add an event
  • off(event, [handler]) - method to remove an event
  • trigger(eventData) - method to trigger an event
That you can use it to add event listeners and trigger events. See Events for additional information.
